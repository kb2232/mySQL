# Installation

1.  First You need to download mysql or simply use cloud 9
2.  On Cloud9:
    >> mysql-ctl start

    >> mysql-ctl cli

    >> mysql-ctl stop

3.  If you have downloaded mysql - then
    >> export PATH=${PATH}:/usr/local/mysql/bin/

    >> mysql -u {user} -p {password};
